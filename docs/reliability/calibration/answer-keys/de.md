# Koordinationsschlüssel — deutsche Kalibrierung

Key ID: `human-calibration-key-de-1.0.0`

Nur zur Schulung und getrennt vom Kodierpaket aufzubewahren. Die Bereiche
steuern das Gespräch; sie sind keine akzeptierten Projektannotationen.

| Item | Entwurfsrolle | Erwartbarer Begründungsbereich |
|---|---|---|
| `calibration-synthetic-de-01` | positive Identifikation | `mipvu_indirect`; exakte Grenze `Fenster`; politische Möglichkeit wird als räumlich-visuelle Öffnung verstanden. Konventioneller Sprachgebrauch ist eine gültige Alternativlesart. |
| `calibration-synthetic-de-02` | negative Kontrolle | `non_metaphor`, exakte Grenze; kontextuelle und grundlegende Bedeutung bezeichnen dasselbe materielle Bauteil. |
| `calibration-synthetic-de-03` | mehrdeutige Identifikation | `mipvu_indirect`, `mipvu_personification` oder `uncertain` sind vertretbar, wenn körperliches Tragen, Fortdauer und der Erinnerung zugeschriebene Agency getrennt geprüft werden. |
| `calibration-synthetic-de-04` | CMT | Quelle: behinderter Weg/Aussaat; Ziel: soziale Einigung. Hindernisse der Einigung werden als Hindernisse auf einem Weg verstanden. Die Alternativlesart soll Bildmischung oder konventionelle Wendung behandeln. |
| `calibration-synthetic-de-05` | Interpretation | Verpflichtung und Gewaltlogik sind vorhanden. Der Schrein stützt wahrscheinlich heiliges Objekt und Reinigung. `sacrificial_body` ist vorhanden oder unsicher, je nachdem ob `Blut vergießen` als verlangter körperlicher Einsatz statt nur als Gewalt gelesen wird. |
| `calibration-synthetic-de-06` | Agency/Absenz | Der Erlass ist grammatisches Agens, Ordnung Ergebnis/Patiens. Menschliche Autorität ist unterdrückt; bei nur einem Satz ist `uncertain` meist angemessener als eine weitreichende Absenzbehauptung. |

Remediation ist nötig, wenn eine Übersetzung den Ausgangstext ersetzt, jede
Gewalt automatisch als Opfer gilt oder aus unzureichendem Umfang eine breite
Absenz gefolgert wird.

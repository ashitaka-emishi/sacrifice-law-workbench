# British WWI / Lloyd George Corpus

This draft corpus contains selected English OCR source text from two
public-domain Internet Archive volumes:

- David Lloyd George, *Through Terror to Triumph* (1915)
- David Lloyd George, *The Great Crusade* (1918)

Regenerate raw files with:

```bash
python3 scripts/extract-wwi-britain-speeches.py
```

The raw files are committed because the source editions are public domain. The
text remains OCR-derived and requires source-edition verification and OCR review
before publication-grade historical claims.

# Blind CMT Field Review v1

Treat each supplied span as a bounded classification exercise, not evidence that
the project has accepted a metaphor. Record source and target domains, the
conceptual mapping, entailments, cluster, confidence, uncertainty, and a
plausible rival reading. Use controlled vocabulary IDs where the schema requires
them and preserve the source-language span separately from any English gloss.

Return only fields defined by submission schema version 1.0.0.

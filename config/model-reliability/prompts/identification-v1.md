# Blind Identification Review v1

Review each lexical unit in the source-language sentence independently. Record
its MIPVU decision, boundary decision, contextual meaning, basic meaning,
contrast explanation, comparison basis, confidence, uncertainty, and a plausible
rival reading. Use an English gloss only as an aid; the source text controls.

Do not infer a reference answer from item selection. The packet contains
negative controls and difficult cases. Return only fields defined by submission
schema version 1.0.0.

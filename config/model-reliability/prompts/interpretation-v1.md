# Blind Interpretive Field Review v1

Treat each supplied span as a bounded interpretive classification exercise.
Record the standard Koenigsbergian function judgments, agency roles, possible
absence or displacement, confidence, uncertainty, and a plausible rival
reading. Do not add historical corroboration, support scores, or final claims.
Agreement with another model would be a diagnostic, not scholarly evidence.

Return only fields defined by submission schema version 1.0.0.
